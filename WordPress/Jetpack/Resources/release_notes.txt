For this month's release, we've:
- Added support for Image Playgrounds! Use it to add images to your posts, generate a featured image, or site icon.
- Added undo and redo to the experimental editor (have you tried it yet?)
- Made the Gravatar Quick Editor a bit nicer.
- Replacing the featured image for a post is faster now.
- Viewing images in your media library is a little nicer now.
- We did the usual bug fixes and performance improvements (mostly in the editor)
